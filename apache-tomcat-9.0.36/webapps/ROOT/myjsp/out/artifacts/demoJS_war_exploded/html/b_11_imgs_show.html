任务11. 制作图片展示效果

1．训练要点：

 掌握获取和设置属性的方法attr()/prop()
 掌握创建节点和插入节点的方法
 掌握html()方法

2．需求说明

在页面中有四幅小图片和一个图片展示区域，当鼠标移入某幅小图片时，在图片展示区域可看到其对应的大图。
为凸显当前的小图状态，把其它三幅小图透明度设为0.5。

完成结果


实现代码 
$(function () {
    $('.thumbs a').addClass("alpha");
    $('.thumbs a:lt(1)').removeClass("alpha");
    $('.thumbs a').mouseover(function () {
        $('#largeImg').attr("src",this);
        $(this).removeClass("alpha");
        $(this).siblings().addClass("alpha");
    })
})
